# miniprogram-3

这是一个微信小程序项目，主要包含以下功能：

## 功能模块

- 首页：主要功能展示
- 发现：内容发现页面
- 我的：用户中心页面

## 项目结构

```
miniprogram/
├── pages/          # 页面文件
│   ├── index/      # 首页
│   ├── discover/   # 发现页
│   └── profile/    # 个人中心
├── images/         # 图片资源
└── components/     # 公共组件
```

## 开发环境

- 微信开发者工具
- Node.js
- 微信云开发
